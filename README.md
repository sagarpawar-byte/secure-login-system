# Secure Login System

A secure login system built using Flask and SQLite.

## Features

* User Registration
* Secure Login
* Password Hashing using Bcrypt
* Session Management
* Logout Functionality
* SQL Injection Protection

## Technologies Used

* Python
* Flask
* SQLite
* Flask-Bcrypt

## Project Structure

secure-login-system/
│
├── app.py
├── database.db
├── requirements.txt
├── README.md
│
└── templates/
├── login.html
├── register.html
└── dashboard.html

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
